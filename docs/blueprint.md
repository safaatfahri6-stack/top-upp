# **App Name**: TopUpGo

## Core Features:

- Google Sign-In: Enable users to sign in securely using their Google accounts with the provided client ID.
- Balance Display: Display the user's current balance.
- Top-Up Options: Present various top-up amount options for the user to choose from.
- Payment Processing: Integrate a secure payment gateway to process the selected top-up amount.
- Transaction History: Show a history of the user's top-up transactions.

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2) to inspire trust and security.
- Background color: Light gray (#F5F5F5) for a clean, modern feel.
- Accent color: Green (#2ecc71) for success messages and call-to-action buttons.
- Body and headline font: 'Inter', a sans-serif font for a modern, readable experience.
- Use simple, line-based icons to represent different payment methods and actions.
- Clean, minimalist layout with clear calls to action.
- Subtle animations for loading states and successful transactions.
